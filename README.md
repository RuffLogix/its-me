# Contributor Translation Guide

Help us translate the phrase below into more languages:

`Hello, it's me!`

## How to contribute

1. Create a new file named with the language code, for example `es.md`, `fr.md`, or `ja.md`.
2. Put only one line in that file: the translation of `Hello, it's me!`.
3. Keep punctuation and tone natural for the target language.
4. Open a pull request with your new language file.

## Current languages

- `en.md`: `Hello, it's me!`
- `th.md`: `สวัสดีครับ ผมเอง!`
